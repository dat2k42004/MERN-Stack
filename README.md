1. Frontend
   +	Tạo dự án React + Vite: npm create vite@latest
   +	Cài đặt: npm install
   +	Cài đặt các thư việc mở rộng: npm antd axios moment react-dom react-redux react-router-dom
   +	Chạy ứng dụng: npm run dev
2. Backend
   +	Cài đặt Express JS: npm install express
   +	Cài đặt các thư viện mở rộng: npm install cors dotenv jsonwebtoken nodemon bcryptjs mongoose
